# Luby


Teste da Luby para Desenvolvedor Jr.

### Deploy
Feito Deploy no Vercel
Link: https://luby-kenzie-teste.vercel.app/
### Prerequisites

NodeJS
Yarn ou NPM



### Installing

Usando o yarn:

```
yarn install
```
E depois, para rodar:
```
yarn start
```
